入门指南

Tripo 是 VAST 公司开发的 AI 驱动 3D 建模平台，能够在数秒内将文字描述和图片转换为高质量、可直接投入生产的 3D 模型。基于先进的 3D 重建大模型技术，Tripo 帮助创作者、开发者和企业生成具有精细几何结构、PBR 物理材质和自然纹理的规范化 3D 资产——无需任何 3D 建模专业知识。

无论您是在开发游戏、创建 AR/VR 体验，还是进行产品原型设计，Tripo 都能将传统需要数小时的 3D 创作流程缩短至几分钟。我们提供 Tripo Web（直观的网页端界面）和 Tripo API（企业级接入方案），灵活适配您的工作流程。
Tripo 提供一整套由 AI 驱动的 3D 创作工具：

核心生成能力
  • 文本生成 3D：通过自然语言描述直接生成 3D 模型
  • 图片生成 3D：将单张图片或草图转换为 3D 资产
  • 多视图生成 3D：基于 2–4 张参考图生成更高精度的 3D 模型
支持不同行业需求
  • 高精度模型：最高支持 200 万面，适用于 3D 打印与视觉艺术
  • 智能拓扑网络：2 秒极速生成干净、规范的布线
3D全流程覆盖
  • 专业级3D生图：平台内集成顶级 AI 生图模型
  • 纹理生成与编辑：自动生成高质量材质贴图，支持二次编辑
  • 部件智能拆分：自动识别并拆分模型为独立部件
  • 重拓扑：支持自定义面数，一键减面或切换为四边面
  • 绑骨与动画：一键添加标准骨骼结构
  • 访问平台：前往 studio.tripo3d.com 注册/登录
  • 选择模式：文生3D（输入描述）或图生3D（上传照片/草图）
  • 生成模型：点击"生成"——AI 在 10 秒至 2 分钟内完成建模
  • 优化编辑：使用"编辑"功能进行自动骨骼绑定、网格精修或提升纹理分辨率
  • 导出使用：选择格式（GLB/FBX/OBJ）下载完整 PBR 贴图包
Tripo 目前支持 6 种导出格式：
  • GLB – 适用于 Web、AR/VR 和游戏引擎（内嵌纹理）
  • USD – 通用场景描述格式，适用于影视特效、动画
  • FBX – 游戏和 3D 动画软件的行业标准
  • OBJ – 通用 3D 编辑和基础 3D 打印兼容性格式
  • STL – 3D 打印标准格式（仅几何体）
  • 3MF – 新一代 3D 打印格式，支持色彩和纹理信息

积分重置
月度订阅积分在每月的计费日重置。例如 3 月 15 日订阅，积分将在每月 15 日重置。上一周期未使用的月度积分不会结转。
积分类型是否过期规则
月度订阅积分会每月计费日重置，未用完不结转
单独购买的额外积分不会永久有效，用完为止
如果订阅过期或取消，所有未使用的月度订阅积分将被清零。但单独购买的额外积分永久保留。
异常与退款
不会。仅在任务状态为"成功"时才扣除积分。如生成失败不消耗任何积分，可重新尝试。

插件教程


Bridge 简介

Tripo3D Blender Bridge 是一个轻量级的附加组件，可将 Blender 直接连接到 Tripo studio 首页。通过此 Bridge，您可以将浏览器中生成的模型实时直接发送到 Blender，无需手动下载、导入或额外的步骤。它旨在简化艺术家和设计师的工作流程，让您专注于创作而非文件处理。

系统要求

  • 操作系统： Windows, Mac
  • Blender： 4.1.0 或更高版本
  • 浏览器：
  • 支持 Google Chrome 116+、Edge 116+、Opera 102+
  • 其他浏览器不支持
  • 默认导出设置：
  • 打包UV: 否
  • 导出顶点色: 否
  • 底部中心轴点: 是
  • 纹理贴图：当前最大值

    • 分步安装

    步骤 1： 访问 Tripo Studio 并下载Tripo3D Blender Bridge ZIP 文件。


    步骤 2： 在 Blender 中，点击Edit → Preferences → Add-ons。


    步骤 3：

  • 点击 Install from Disk，找到下载的ZIP 文件（不要解压），然后启用该插件。

  • 然后按 N 键检查 Tripo bridge 面板。

    • 步骤 4：

  • 打开 Tripo Studio 网站，点击发送至3D软件（DCC Bridge） 按钮，并激活 Blender 插件。
  • 确认网页和插件连接成功。

  • 在小窗口中点击 导出 按钮。

  • 如果您想导出动画，在导出面板中选择您要导出的动画，然后点击发送至 发送到指定的软件。

    • 步骤 5： 现在可以在 Blender 中看到从网站发送过来的模型了。


    常见问题

    问：我能否同时向多个 Blender 传输数据？

    答： 不能。连接只识别第一个启动的 Blender 实例。打开多个实例将导致端口被占用的错误。

    问：我更新了插件，但新的 Tripo Bridge 显示“All connections closed”并且无法连接。我该怎么办？

    答： 重启 Blender 并在 Add-ons 首选项中重新激活插件。确保 Tripo Bridge Log 显示“Server running on port [XXXX]”以确认连接成功。


    Tripo3D Unity Bridge 是一款轻量级插件，可将 Unity 直接连接到 Tripo Studio 首页。通过此 Bridge，您可以将生成的模型从浏览器直接发送到 Unity 编辑器中，无需手动下载、导入或额外的步骤。

    它旨在简化艺术家、技术美术师和游戏开发者的工作流程，让您能够专注于场景搭建和游戏玩法集成，而不是文件管理。

    要求

  • 操作系统： Windows、Mac
  • Unity： 2021.3 LTS 或更高版本（推荐 2022.3 LTS+ 或 Unity6 LTS+）
  • 浏览器：
  • 支持 Google Chrome 116+、Edge 116+、Opera 102+
  • 不支持其他浏览器
  • 默认导入设置：
  • 生成碰撞体 (Generate Colliders)：`false`
  • 导入材质 (Import Materials)：`true`
  • 导入纹理 (Import Textures)：`true`
  • 枢轴点 (Pivot)：底部中心 (Center Bottom)
  • 纹理分辨率 (Texture Resolution)：当前最大值 (Current Maximum)

    • 分步安装

    第 1 步： 访问 Tripo Studio 并下载Tripo3D Unity Bridge ZIP 文件。


    第 2 步：

  • 将包导入 Unity。打开您的 Unity 项目并导航至：Window → Package Manager（窗口 → 包管理器）

  • 在包管理器窗口中，点击左上角的 “+” 按钮。选择Add package from disk…（从磁盘添加包…）

  • 浏览到解压后的插件文件夹。选择 `package.json` 文件。

  • 安装完成后，插件将出现在您的包管理器列表中，表示将在项目中可用。

    • 第 3 步：

  • 安装后：导航至：Tools → Tripo Bridge（工具 → Tripo Bridge）

  • 面板打开后，Bridge 服务会自动启动。
  • Bridge 服务在连接到网页浏览器时必须保持运行。
  • 在模型传输过程中，请勿关闭 Unity 或 Bridge 面板。
  • 在 Tripo Bridge 面板内，渲染管线会根据当前的 Unity 项目自动检测。支持的管线包括：
  • Standard（内置渲染管线）
  • URP
  • HDRP

    • 您还可以查看连接状态和详细的日志输出。


    附注：如果您需要手动停止服务，请点击面板中的 Stop Server（停止服务器）。但是，要从 Tripo Studio 发送或接收模型，服务必须正在运行。如果服务已停止，请在尝试传输之前点击Start Server（启动服务器）。


    第 4 步：

  • 打开 Tripo Studio 网站，点击发送至3D软件（DCC Bridge） 按钮，并激活 Unity 插件。
  • 确认网页和插件连接成功。

  • 选择一个模型。点击小窗口中的 Export（导出）按钮。
  • 点击 Send to Unity（发送到 Unity）。

    • 传输后，请确保 Unity 窗口处于聚焦状态，以便 Unity UI 能够及时刷新并正确显示传入的模型。

  • 如果您想导出带动画的模型，请先在 Tripo Studio 中生成动画，然后在导出面板中选择所需的动画并将其直接发送到 Unity。

    • 第 5 步： 您现在可以在 Unity 中看到从网站发送的模型。


    常见问题

    问：导入的模型在 Unity 中显示为紫色。

    答： Unity 中出现的紫色（品红色）材质通常表示着色器或渲染管线不匹配。

    如果您的项目使用 URP 或 HDRP：

  • 打开 Tripo Bridge 面板。
  • 确保选择了正确的渲染管线（URP / HDRP），然后再次从 Tripo Studio 发送模型。
  • 另外，您可以手动修复材质：选择受影响的材质，将其着色器更改为 Universal Render Pipeline/Lit（适用于 URP）或 HDRP/Lit（适用于 HDRP），替换相应的纹理；如果问题仍然存在，请确保在 Project Settings → Graphics（项目设置 → 图形）中分配了有效的渲染管线资源 (Render Pipeline Asset)。

    • 问：我可以同时将数据传输到多个 Unity 吗？

    Tripo3D Unreal Engine Bridge 是一款轻量级插件，可将 Unreal Engine 直接连接到 Tripo Studio 首页。

    通过此 Bridge，您可以将生成的模型实时从浏览器直接发送到您的 Unreal 项目中——无需手动下载或导入。它简化了艺术家和游戏开发者的工作流程，让您能够专注于场景构建和游戏玩法，而不是文件处理。
